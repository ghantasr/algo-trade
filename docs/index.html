How It Works

📡 Scan NSE Stocks → 📲 BUY Alert on Telegram → 👤 You buy on broker → ✅ Confirm with /bought → 🤖 Bot tracks position → 🔴 SELL Alert sent → 💰 You sell & confirm

Telegram Commands

CommandAction
/runStart scanning NSE stocks
/stopPause scanning
/scanScan for buy opportunities now
/bucketShow holdings & P&L (after fees)
/bought SYMBOL QTY PRICEConfirm you bought a stock
/sold SYMBOL QTY PRICEConfirm you sold a stock
/premarketRun pre-market analysis
/helpShow all commands

Quick Start

# Clone the repo
git clone https://github.com/ghantasr/algo-trade.git
cd algo-trade

# Set up environment
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env

# Edit .env with your Telegram credentials
# Then run:
python main.py

Send /run on Telegram during market hours (9:15 AM – 3:30 PM IST).

Features

  • MA Crossover Strategy — Moving average based buy/sell signals
  • Momentum Scanner — Detects high-volume breakout stocks
  • Pre-Market Analysis — Analyze market sentiment before open
  • Paper Trading — Track positions with a virtual "bucket"
  • Fee Calculator — Zerodha-style brokerage fees included
  • Exit Advisor — Smart exit recommendations with stop-loss
  • Market Hours — Auto-pauses outside NSE hours

Disclaimer

This is an educational tool, not financial advice. Intraday trading involves significant risk. Never invest money you cannot afford to lose. The bot does not place real orders — you trade manually on your broker platform.
